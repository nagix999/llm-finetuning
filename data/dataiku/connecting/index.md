Connecting to data[Â¶](#connecting-to-data "Permalink to this heading")
======================================================================


The first task when using Data Science Studio is to define *datasets* to connect to your data sources.


A dataset is a series of records with the same schema. It is quite analogous to a table in the SQL world.


For a more global explanation about the different kinds of datasets, see the [DSS concepts](../concepts/index.html) page.



See also


For more information, see the [Concept \| Data connections](https://knowledge.dataiku.com/latest/data-sourcing/connections/concept-connections.html) article in the Knowledge Base.
